# DMBSearchViewer Glossary

## Project-specific section

When copying this file to another PageBuilder ecosystem project, update this section first.

- Project name: `DMBSearchViewer`
- Project role: runtime search UI and provider aggregation package.
- Publication host: `labs_idemobi_com`

## DMBSearchViewer

Package that exposes search UI, search routes, and provider aggregation for host MVC applications.

## Search Provider

Implementation of `ISearchProvider` that returns results for one search source.

## Composite Agent

Runtime component that queries configured providers and combines their results and errors into a response.

## Search Query

Input model describing the user search text and related request options.

## Search Result

Returned item containing the searchable page or documentation result displayed by the result page.

## Provider Error

Structured error returned when one provider fails without breaking the full search response.

## Navbar Search Component

Reusable UI component that adds search entry behavior to the host navigation experience.

## SearchBuilder Database

SQLite database generated by `DMBSearchBuilder` and consumed by `SearchBuilderSearchProvider`.

## Documentation Provider

Search provider that exposes generated documentation records from `DMBDocumentationViewer`.
